# 名言卡片生成器 - 页面设计规范（Desktop First）

## 1. 全局设计（Global Styles）
- 设计基调：简洁、内容突出、输入与预览并列对照。
- 背景色：#F7F8FA（页面背景），#FFFFFF（卡片/面板背景）。
- 主色（Accent）：#3B82F6（按钮/高亮/聚焦边框）。
- 文字颜色：#111827（主文字），#6B7280（次要说明/占位）。
- 圆角：12px（面板、预览卡片）；输入框 10px。
- 阴影：轻阴影（例如 0 8px 24px rgba(0,0,0,0.08)）用于预览卡片。
- 排版（Typography）：
  - H1：24/32 semibold
  - 区块标题：16/24 semibold
  - 正文：14/20 regular
  - 表单标签：12/18 medium
- 交互状态：
  - Button：默认主色填充；hover 加深 6–8%；disabled 降低不透明度。
  - Input：focus 显示主色描边与轻微外发光；error 显示红色提示与描边。

## 2. 页面：名言卡片生成器预览页

### 2.1 Meta Information
- Title：名言卡片生成器 - 预览
- Description：输入名言与作者，在右侧实时查看卡片预览。
- Open Graph：
  - og:title：名言卡片生成器
  - og:description：基础 UI 预览界面：输入 + 预览占位图。

### 2.2 Layout（布局）
- Desktop（>=1024px）优先：双栏布局。
  - 使用 CSS Grid：`grid-template-columns: 420px 1fr; gap: 24px;`
  - 左侧输入区宽度固定（或最大 480px），右侧预览自适应。
- Tablet（768–1023px）：双栏可保持，但左侧宽度缩小（例如 360px）。
- Mobile（<768px）：改为单列堆叠：先输入区，后预览区；间距 16px。
- 页面容器：最大宽度 1200px，左右内边距 24px，居中。

### 2.3 Page Structure（页面结构）
1. 顶部标题区（可选极简，无复杂导航）
2. 主内容区（左右分栏）
   - 左：输入面板
   - 右：预览面板

### 2.4 Sections & Components（区块与组件）

#### A. 顶部标题区
- 元素：
  - 页面标题："名言卡片生成器"
  - 简短说明："在左侧输入内容，右侧查看预览"
- 布局：标题与说明垂直排列，底部留白 16px。

#### B. 左侧输入区（Input Panel）
- 容器：白底卡片面板 + 16–20px 内边距 + 圆角。
- 表单组件（基础必需）：
  1. 名言正文输入（Textarea，多行）
     - Label："名言正文"
     - Placeholder："请输入名言内容…"
     - 校验：允许为空但会影响预览；为空时在下方显示提示文案（弱提示）。
  2. 作者/来源（Input，单行，可选）
     - Label："作者/来源（可选）"
     - Placeholder："例如：鲁迅"
  3. 更新预览（Primary Button）
     - 文案："更新预览"
     - 交互：点击后驱动右侧预览内容刷新；若实现为实时更新，可保留按钮作为手动刷新入口。
- 细节：
  - 表单项间距 12–14px；按钮与表单间距 16px。
  - 键盘可用性：Tab 顺序从正文 -> 作者 -> 更新预览。

#### C. 右侧预览区（Preview Panel，占位图）
- 容器：白底卡片面板 + 16–20px 内边距 + 圆角。
- 结构：
  1. 预览标题行
     - 文案："预览"
     - 右侧可放状态标签（例如“占位预览”/“已更新”），如不实现可省略。
  2. 预览画布区域（占位图）
     - 固定比例建议：1:1 或 4:5（任选其一，但需保持稳定，避免跳动）。
     - 占位状态：
       - 灰色底（例如 #EEF2F7）
       - 居中文案："预览将在这里显示"
     - 更新后状态（仍可为简化占位实现）：
       - 在占位图上以居中方式叠加名言正文与作者
       - 文本层级：正文更大、更粗；作者较小、靠下或右下
- 交互：仅展示，不要求编辑；滚动不应影响左侧输入的可见性（可选：右侧预览区域内部滚动）。

### 2.5 Responsive Behavior（响应式补充）
- Mobile 堆叠时：
  - 输入区在上，预览区在下；预览画布高度适配屏宽。
- 字体与间距随断点缩放：移动端内边距 16px，标题字号可降到 20/28。

### 2.6 动效与过渡（可选，轻量）
- 更新预览：预览画布内容淡入（150–200ms opacity transition）。
- 输入 focus：描边与阴影过渡（120ms）。
